# Usage

Try prompts like:
- "AzureCoach: audit the repo for an Azure deployment plan and list next steps."
- "AzureCoach: create a Bicep module to host the frontend and a CI/CD workflow."
- "AzureCoach: teach me how Entra ID and Managed Identity work, and show code examples."

If you want the agent to run commands or create sub-agents, reply with explicit approval and any required credentials or details.
